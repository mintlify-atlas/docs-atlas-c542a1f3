This guide will walk you through creating your first workspace, page, and content in AppFlowy. By the end, you'll understand the basics of navigating and using AppFlowy.

<Note>
  **Prerequisites**: Make sure you've [installed AppFlowy](/getting-started/installation) on your device.
</Note>

## Get Started in 4 Steps

<Steps>
  <Step title="Launch AppFlowy and choose your workspace type">
    When you first open AppFlowy, you'll be presented with two workspace options:

    <Tabs>
      <Tab title="Cloud Workspace (Recommended)">
        **Best for**: Collaboration, syncing across devices, cloud AI features

        - Sign in with your email or Google account
        - Data syncs across all your devices
        - Enable real-time collaboration with team members
        - Access to cloud-powered AI features

        Click **Sign In** or **Create Account** to get started.
      </Tab>

      <Tab title="Vault Workspace">
        **Best for**: Privacy-focused users, offline work, local AI

        - Completely offline and private
        - No data leaves your device
        - AI runs locally with your own models
        - Perfect for sensitive data

        Click **Create Vault Workspace** to continue without an account.
      </Tab>
    </Tabs>

    <Tip>
      You can create multiple workspaces and switch between them anytime. Start with a Cloud Workspace if you're unsure.
    </Tip>
  </Step>

  <Step title="Create your first page">
    Your workspace opens with a default view. Let's create your first page:

    1. Click the **+** button in the sidebar or press `Cmd/Ctrl + N`
    2. Choose a page type:
       - **Document** - Rich text notes with blocks and embeds
       - **Grid** - Spreadsheet-like database
       - **Board** - Kanban board for tasks
       - **Calendar** - Date-based organization
    3. Give your page a title (e.g., "My First Page")
    4. Press `Enter` to create the page

    <img src="/images/create-page.png" alt="Creating a new page" />

    <Note>
      Pages are the building blocks of AppFlowy. Think of them as flexible documents that can contain anything from simple notes to complex databases.
    </Note>
  </Step>

  <Step title="Add content to your page">
    Now let's add some content. AppFlowy uses a block-based editor where you can add different types of content:

    ### Text and Formatting
    - Type `/` to open the block menu
    - Start typing to see available blocks
    - Use markdown shortcuts:
      - `**bold**` for **bold text**
      - `*italic*` for *italic text*
      - `# ` for headings
      - `- ` for bullet lists
      - `[ ] ` for checkboxes

    ### Common Blocks to Try
    <CardGroup cols={2}>
      <Card title="Heading" icon="heading">
        Type `/heading` or use `#`, `##`, `###`
      </Card>
      <Card title="Checklist" icon="square-check">
        Type `/todo` or `[ ]` for task lists
      </Card>
      <Card title="Code Block" icon="code">
        Type `/code` for syntax-highlighted code
      </Card>
      <Card title="Table" icon="table">
        Type `/table` to insert a table
      </Card>
    </CardGroup>

    ### Try AI Features
    AppFlowy includes built-in AI assistance:

    1. Select any text
    2. Click the **AI** button that appears
    3. Choose an action:
       - Improve writing
       - Fix spelling & grammar
       - Make shorter/longer
       - Translate
       - Summarize

    <Tip>
      Press `Cmd/Ctrl + /` to quickly open the AI assistant anywhere in your document.
    </Tip>
  </Step>

  <Step title="Organize your workspace">
    As you create more pages, keep them organized:

    ### Create folders
    1. Right-click in the sidebar
    2. Select **New Folder**
    3. Give it a name (e.g., "Projects", "Personal")
    4. Drag pages into folders

    ### Use favorites
    - Hover over any page in the sidebar
    - Click the **star icon** to add to favorites
    - Access favorites quickly from the top of your sidebar

    ### Search your workspace
    - Press `Cmd/Ctrl + P` to open quick search
    - Type to search by page title
    - Use AI-powered search for natural language queries

    <img src="/images/organize-workspace.png" alt="Organizing pages in folders" />
  </Step>
</Steps>

- ✅ Use AI features to enhance your writing

## Quick Tips

<AccordionGroup>
  <Accordion title="Keyboard Shortcuts" icon="keyboard">
    Essential shortcuts to boost your productivity:

    | Action | Windows/Linux | macOS |
    |--------|--------------|-------|
    | New page | `Ctrl + N` | `Cmd + N` |
    | Quick search | `Ctrl + P` | `Cmd + P` |
    | Block menu | `/` | `/` |
    | AI assistant | `Ctrl + /` | `Cmd + /` |
    | Toggle sidebar | `Ctrl + \` | `Cmd + \` |
    | Delete page | `Ctrl + Backspace` | `Cmd + Backspace` |

    View all shortcuts by pressing `Ctrl/Cmd + /` then typing "shortcuts".
  </Accordion>

  <Accordion title="Mobile Apps" icon="mobile">
    AppFlowy works seamlessly across devices:

    - Download the [iOS app](https://apps.apple.com/app/appflowy/id6457261352) or [Android app](https://play.google.com/store/apps/details?id=io.appflowy.appflowy)
    - Sign in with the same account
    - Your workspace syncs automatically
    - Add widgets for quick access to pages

    See [Mobile Setup](/mobile/ios) for platform-specific features.
  </Accordion>

  <Accordion title="Templates" icon="file-template">
    Jump-start your work with templates:

    1. Click the **+** button to create a new page
    2. Select **Use Template**
    3. Browse categories:
       - Project management
       - Meeting notes
       - Personal planning
       - Knowledge base
    4. Click a template to create a pre-formatted page

    Learn more about [Templates](/core/templates).
  </Accordion>

  <Accordion title="Collaboration" icon="users">
    Invite others to collaborate:

    1. Click **Share** in the top-right corner of any page
    2. Enter email addresses
    3. Set permissions:
       - **Can view** - Read-only access
       - **Can edit** - Full editing access
    4. Click **Send Invite**

    Recipients will receive an email invitation. Learn more about [Collaboration](/collaboration/sharing).
  </Accordion>
</AccordionGroup>
