## Overview

AppFlowy provides a comprehensive suite of features designed to help you work more efficiently while maintaining complete control over your data. From rich document editing to powerful databases, AI assistance, and real-time collaboration, AppFlowy has everything you need for modern knowledge work.

<Note>
All features work seamlessly across desktop and mobile platforms with real-time synchronization
</Note>

### Database Properties

AppFlowy supports a comprehensive set of field types:

- **Text**: Single and multi-line text fields
- **Number**: Various formats including currency, percentage, and custom formats
- **Date**: Date and time with range support and timezone handling
- **Select & Multi-select**: Dropdown options with 10 premium colors for Pro users
- **Checkbox**: Binary yes/no fields
- **URL**: Clickable web links
- **Files & Media**: Upload and attach files directly to records
- **Checklist**: Task lists with progress tracking
- **Relation**: Link records between databases
- **Created At / Updated At**: Automatic timestamps
