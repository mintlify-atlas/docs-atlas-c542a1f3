Mentions in AppFlowy let you bring someone's attention to specific content, ask questions, or assign tasks. Tagged users receive email notifications so they never miss important updates.

## Overview

With mentions, you can:
- **Notify teammates** about specific content or changes
- **Assign tasks** by mentioning someone in context
- **Ask questions** directly in documents
- **Link to pages** using page mentions
- **Set reminders** with date mentions
- Receive **email notifications** when mentioned

<Note>
Mention features were introduced in version 0.9.7 for both desktop and mobile platforms.
</Note>

## Mentioning People

### Creating a Person Mention

There are two ways to mention someone:

<Steps>
  <Step title="Use @ Symbol">
    Type `@` anywhere in your document. A menu appears showing workspace members.
  </Step>
  
  <Step title="Search and Select">
    Start typing the person's name to filter the list, then click to select them.
  </Step>
  
  <Step title="Continue Writing">
    The mention appears as a highlighted tag. Continue writing your message or context.
  </Step>
</Steps>

**Alternative Method**: Use the slash menu:

1. Type `/` to open the command menu
2. Select "Mention a person" or type "@"
3. Choose the person from the list

<Info>
You can mention any member of your workspace. To mention external collaborators, first invite them as [Guest Editors](/collaboration/guest-editors).
</Info>

### Email Notifications

When you mention someone:

- They receive an **email notification** with:
  - Who mentioned them
  - The page where they were mentioned
  - Context around the mention
  - A direct link to the mention location

- Notifications are sent immediately when the mention is created
- Recipients can click the link to jump directly to the mentioned content

<Note>
Email notifications are sent to the email address associated with the user's AppFlowy account.
</Note>

## Using Mentions for Task Assignment

Mentions are perfect for assigning work to teammates:

### Assignment Pattern

Create clear task assignments:

```text
@John Smith - Please review the Q4 budget proposal by Friday

@Sarah Lee - Can you update the client presentation with the latest metrics?

@Team Lead - This section needs your approval before we publish
```

### Best Practices for Assignments

<Steps>
  <Step title="Be Specific">
    Clearly state what needs to be done. Include context and deadlines.
  </Step>
  
  <Step title="Mention One Person">
    Assign tasks to individuals, not groups, for accountability.
  </Step>
  
  <Step title="Add Due Dates">
    Combine person mentions with date mentions for complete task information.
  </Step>
  
  <Step title="Follow Up">
    Check in if the mentioned person hasn't responded or completed the task.
  </Step>
</Steps>

<Warning>
Mentions are notifications, not a formal task management system. For complex project tracking, consider using AppFlowy's database features with assigned person fields.
</Warning>

## Page Mentions

Link to other pages in your workspace using page mentions:

### Creating Page Mentions

<Steps>
  <Step title="Type $ Symbol">
    Use `$` to trigger the page mention menu.
  </Step>
  
  <Step title="Search for Pages">
    Start typing the page name to filter available pages.
  </Step>
  
  <Step title="Select Page">
    Click the page you want to reference. A link is created.
  </Step>
</Steps>

### Page Mention Use Cases

<CardGroup cols={2}>
  <Card title="Reference Documents" icon="link">
    Link to related pages, documentation, or background information.
  </Card>
  
  <Card title="Create Navigation" icon="map">
    Build a table of contents or index page with mentions to other pages.
  </Card>
  
  <Card title="Show Dependencies" icon="diagram-project">
    Indicate which pages are related or depend on each other.
  </Card>
  
  <Card title="Quick Access" icon="bolt">

### Handling Deleted Pages

If a mentioned page is deleted or you lose access:

- The mention shows as **"Deleted Page"** or **"No Access"**
- A trash icon appears for deleted pages
- You can remove the mention or replace it with a valid page

<Info>
Page mentions automatically update if the referenced page is renamed.
</Info>

## Date Mentions

Add dates and create reminders with date mentions:

### Creating Date Mentions

1. Type `$` or `/` to open the command menu
2. Select "Date" or start typing a date
3. Use the date picker to select a specific date
4. Optionally add a time and reminder

### Date Mention Features

- **Date picker**: Visual calendar for selecting dates
- **Natural language**: Type "tomorrow", "next Friday", etc.
- **Time support**: Add specific times to your dates
- **Reminders**: Set notifications for important dates

<Note>
Date mentions support various formats and can be configured to match your preferences in Settings.
</Note>

## Mention Notifications

### Desktop Notifications

On desktop, you'll see mentions in the notification center:

1. Click the notification bell icon
2. View all your mentions
3. Click any mention to jump to its location
4. Mark mentions as read to clear them

### Mobile Notifications

On mobile devices:

- Receive push notifications for new mentions
- View mentions in the mobile notification center
- Tap to navigate directly to the mentioned content

### Managing Notifications

Control mention notifications in Settings:

1. Go to **Settings** → **Notifications**
2. Toggle mention notifications on or off
3. Configure email notification preferences
4. Set notification frequency (immediate, digest, etc.)

<Info>
Notification settings were improved in version 0.9.0 with a revamped notification center.
</Info>

## Mention Workflows

### Review and Feedback

Use mentions for document reviews:

```text
@Reviewer - Does this section accurately reflect our new pricing model?

@Editor - Please check grammar in the introduction
```

Coordinate content creation:


Capture action items:

```text

- @John - Book the conference room for next week
- @Sarah - Send the presentation to stakeholders by Tuesday
- @Mike - Follow up with the client about their feedback
```

## Troubleshooting

### Person Not Showing in Mention Menu

- Verify they're a member of your workspace
- Check if they've accepted their invitation
- Ensure you have an internet connection
- Refresh the page to sync the latest member list

### Mention Notifications Not Received

- Check notification settings in **Settings** → **Notifications**
- Verify email address is correct in user profile
- Check spam/junk folder for mention emails
- Ensure the workspace is syncing properly

### Can't Mention on a Page

- Verify you have **Can Edit** permissions on the page
- Check that you're not in a read-only view
- Ensure you're logged in to your workspace account
